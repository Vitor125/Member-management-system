Sistema de Gerenciamento de Membros

Sistema fullstack de gerenciamento de membros desenvolvido com HTML, CSS, Python e PostgreSQL.
A aplicação permite realizar operações como adição e visualização de membros, com integração entre frontend, backend e banco de dados.

🚀 Tecnologias Utilizadas
HTML5
CSS3
Python
PostgreSQL
Postman

📌 Funcionalidades
Cadastro de novos membros
Visualização de membros cadastrados
Integração com banco de dados relacional
